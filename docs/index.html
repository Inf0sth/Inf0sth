About me

I'm passionate about everything related to computer science. The ability to create complete systems using only text-based configurations that a computer can interpret is simply magical and wonderful.

As a cybersecurity engineer, I like to think that this field is not solely based on technical or administrative aspects. Rather, we should aim to make the entire business ecosystem understandable to every department, fulfilling a complementary vision to improve the security of our systems.

I love programming and configuring systems because it reminds me that all the tools, protocols, and systems we use are ultimately just files we've defined how to interpret; and something as basic in computing as reading and writing has led to tremendous technological advances.

I enjoy learning new things that I can put into practice.
